Git Cheatsheet

一图流速查表 | Generated Template

配置 (Config)

git config --global user.name "name" 设置用户名
git config --global user.email "email" 设置邮箱
git config --list 查看配置

创建 (Create)

git init 初始化仓库
git clone [url] 克隆远程仓库

基础操作 (Basic)

git status 查看状态
git add . 添加所有文件
git add [file] 添加指定文件
git commit -m "msg" 提交更改
git commit --amend 修改最后一次提交

分支 (Branches)

git branch 列出本地分支
git branch -a 列出所有分支
git branch [name] 新建分支
git checkout [name] 切换分支
git switch [name] 切换分支(新)
git checkout -b [name] 创建并切换
git merge [branch] 合并分支
git branch -d [name] 删除分支

远程 (Remote)

git remote -v 查看远程地址
git fetch [remote] 拉取不合并
git pull [remote] 拉取并合并
git push [remote] 推送到远程
git push -u origin master 推送并关联

撤销 (Undo)

git checkout -- [file] 丢弃工作区修改
git restore [file] 丢弃修改(新)
git reset [file] 暂存区回退
git reset --hard HEAD 重置所有修改
git revert [commit] 反转提交

暂存 (Stash)

git stash 暂存修改
git stash list 查看暂存列表
git stash pop 恢复并删除
git stash drop 删除暂存

日志 (Log & Diff)

git log 查看提交历史
git log --oneline 简洁日志
git log --graph 图形化日志
git diff 工作区vs暂存区
git diff --staged 暂存区vs版本库

标签 (Tag)

git tag 列出标签
git tag [name] 新建标签
git push origin [tag] 推送标签
