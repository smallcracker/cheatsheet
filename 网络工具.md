Ping (连通性测试)
基本用法
ping <host>：测试与目标主机（域名或IP）的连通性
ping 8.8.8.8：测试与 Google DNS 的连通性
控制次数
ping -c 4 <host>：发送 4 个数据包后停止 (Linux/Mac)
ping -n 4 <host>：发送 4 个数据包后停止 (Windows)
持续运行
ping -t <host>：持续发送数据包直到手动停止 (Windows)
ping <host>：默认持续发送 (Linux/Mac) 使用 Ctrl+C 停止

Windows 网络配置 (ipconfig)
查看信息
ipconfig：显示基本信息（IP 地址、子网掩码、默认网关）
ipconfig /all：显示完整信息（包括 MAC 地址、DNS 服务器、DHCP 状态等）
维护与修复
ipconfig /flushdns：清除本地 DNS 解析缓存（解决域名解析问题）
ipconfig /release：释放当前获取的 IP 地址
ipconfig /renew：向 DHCP 服务器重新申请 IP 地址

Linux 网络配置 (ip 命令 - 推荐)
查看信息
ip a / ip addr：显示所有网络接口的地址信息
ip -c a：以彩色输出显示信息（更易读）
ip r / ip route：显示内核路由表
接口管理
ip link set eth0 up：启用 eth0 接口
ip link set eth0 down：禁用 eth0 接口
ip addr add 192.168.1.100/24 dev eth0：临时为接口添加 IP 地址
